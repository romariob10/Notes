Это процесс изменения внутренней структуры программы, не затрагивающий её внешнего поведения и имеющий целью облегчить понимание её работы

Code smell - термин, обозначающий код с признаками (запахами) проблем в системе.

Запахи кода — ключевые признаки необходимости рефакторинга:
- Дублирование кода
- Длинный метод
- Большой класс
- Длинный список параметров
- Стрельба дробью
- Комментарии
