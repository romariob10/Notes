- `\` — за данным символом пишется название команды
- `{` — открывает блок (за данным символом пишется набор инструкций к команде)
- `}` — закрывает блок
- `[` — открывает дополнительный блок команды
- `]` — закрывает дополнительный блок команды


_Преамбула_ — это часть кода, которая находится до команды `\begin{document}`. В ней пишется код, который отвечает за те команды, которые можно будет использовать в теле документа. *Код преамбулы, т.е. код, который находится до команды `\begin{document}`, в PDF-файле не отображается!*

_Тело документа_ — это часть кода, которая находится между командами `\begin{document}` и `\end{document}`. В теле документа пишется код, который отвечает за то, что (например, какой символ) отобразится в итоговом PDF-файле.

## Написание преамбулы

### Указание формата страницы
Начинаются так:
```
\documentclass[параметры класса]{класс документа}
```

##### Основные варианты класса документа (вписать в фигурные скобки вместо "класс документа"):
- `article` — наиболее распространённый класс, используется для написания статей, рефератов, создания конспектов и отчётов по лабораторным работам
- `beamer` — используется для создания презентаций
- `book` — используется для создания книг
- `letter` — используется для написания писем
- `memoir` — наиболее общий класс, который включает в себя функции других

##### Основные варианты параметров класса (вписать через запятую в квадратные скобки вместо "параметры класса"):
- `a4paper` — размер страницы (по умолчанию `letterpaper`)
- `12pt` — размер шрифта (по умолчанию `10pt`)
- `landscape` — альбомная ориентация страницы (по умолчанию книжная)
- `twocolumn` — вывод текста в 2 колонки (по умолчанию в 1 колонку)
- `fleqn` — выравнивание формул по левому краю (по умолчанию по центру)
- `leqno` — номера формул пишутся слева (по умолчанию справа)

### Дополнительные настройки

- `\setlength{\parindent}{1.25cm}` — в таком случае отступ нового обзаца будет равен 1,25 см.
- `\renewcommand{\baselinestretch}{1.25}`  — в таком случае междустрочный интервал будет полуторным.

### Импорт библиотек

```
\usepackage[параметры пакета]{название пакета}
```

##### Список основных пакетов с их параметрами:

- `\usepackage{cmap}` — улучшает копирование и поиск русского текста в PDF-документе
- `\usepackage[T2A]{fontenc}` — используется для корректного отображения кириллицы
- `\usepackage[utf8]{inputenc}` — определяет юникодовую кодировку документа с кодом
- `\usepackage[english,russian]{babel}` — улучшает расстановку переносов и сокращений английских и русских слов
- `\usepackage{amsmath,amsfonts,amssymb,amsthm,mathtools}` — набор пакетов для расширения функционала при работе с формулами
- `\usepackage[left=3cm,right=1cm,top=2cm,bottom=2cm]{geometry}` — задаёт поля страницы
- `\usepackage{graphicx}` — расширяет функционал при работе с рисунками
- `\usepackage[colorlinks,linkcolor=blue]{hyperref}` — цвет ссылок в тексте становится синим
- `\usepackage{physics}` — повышает удобство записи физических формул благодаря кратким и понятным командам
- `\usepackage{esvect}` — даёт возможность рисовать красивый значок вектора
- `\usepackage{icomma}` — делает дополнительный пробел после запятой в математическом режиме в PDF-файле, если после неё есть пробел в файле с кодом
- `\usepackage{indentfirst}` — делает красную строку в первом абзаце раздела
- `\usepackage[labelsep=period,justification=centering]{caption}` — форматирует подписи к рисункам и таблицам, согласно ГОСТ

### Указание доп информации
- `\title{заголовок}`;
- `\author{ФИО автора}`;
- `\date{дата}`.
В команду `\date{дата}` можно написать команду `\today`. В таком случае будет отображаться сегодняшний день. 
##### Эти 3 команды добавляются в преамбулу для того, чтобы в теле документа с помощью команды `\maketitle` можно было создать простейший вариант титульного листа.

## Тело
Напомним, что эта часть документа находится между командами `\begin{document}` (идёт сразу после преамбулы) и `\end{document}`.

##### Когда какая-то часть кода находится между командами `\begin{название окружения}` и `\end{название окружения}`, то говорят, что эта часть кода находится _внутри окружения_ (название окружения). Например, тело документа находится внутри окружения `document`.

```tikz 
\usepackage{pgfplots} 
\pgfplotsset{compat=1.16} 
\begin{document} 
\begin{tikzpicture} 
\begin{axis}[colormap/viridis] 
\addplot3[ surf, samples=18, domain=-3:3 ] {exp(-x^2-y^2)*x}; 
\end{axis} 
\end{tikzpicture} 
\end{document} 
```

Благодаря установленному пакету `indentfirst`, первый абзац каждого раздела будет начинаться с красной строки. Чтобы сделать новый абзац с красной строки внутри раздела, необходимо пропустить 1 строчку в файле с кодом. 

Чтобы новый абзац начался без красной строки, необходимо также пропустить 1 строчку, а в начале следующей — написать команду `\noindent`.

Изначально выравнивание текста идёт по ширине (внутри заданных полей) в случае, если текст доходит до конца строчки, и по левому краю — если не доходит. Чтобы текст был выровнен по центру страницы, необходимо поместить его внутрь окружения `center`. Например:

``` LaTex
\begin{center}
				    Текст, выровненный по центру страницы
\end{center}
```

##### Стандартные варианты **выделения текста**: полужирный, курсив и подчёркнутый реализуются с помощью следующих команд:

- `\textbf{текст, который вы хотите сделать полужирным}`;
- `\textit{текст, который вы хотите выделить курсивом}`;
- `\underline{текст, который вы хотите подчеркнуть}`.

##### Для работы с **размером шрифта** используются следующие команды (от большего к меньшему):

- `{\Huge шрифт, размер которого намного больше стандартного}`;
- `{\huge шрифт, размер которого сильно больше стандартного}`;
- `{\LARGE шрифт, размер которого значительно больше стандартного}`;
- `{\Large шрифт, размер которого немного больше стандартного}`;
- `{\large шрифт, размер которого чуть больше стандартного}`;
- `{\small шрифт, размер которого чуть меньше стандартного}`;
- `{\footnotesize шрифт, размер которого немного меньше стандартного}`;
- `{\scriptsize шрифт, размер которого значительно меньше стандартного}`;
- `{\tiny шрифт, размер которого намного меньше стандартного}`.

##### Для **регулировки размера пропуска** (пробела) используются следующие команды:
- `\vspace{1cm}` — вертикальный пробел, равный 1 см;
- `\hspace{1cm}` — горизонтальный пробел, равный 1 см;
- `\` — одиночный горизонтальный пробел;
- `\\` — прерывание строки на месте данной команды (текст на этой строке будет выровнен по левому краю) и перенос всего остального на следующую;
- `\linebreak` — прерывание строки на месте данной команды (текст на этой строке будет выровнен по её ширине) и перенос всего остального на следующую;
- `\newpage` — переход на новую страницу.

##### Способы поставить **дефис или тире**:
- `-` — дефис;
- `--` — короткое тире;
- `---` — длинное тире.

**Чтобы в текст добавить ссылку на сайт**, необходимо использовать команду `\href`. В первой фигурной скобке пишется полная ссылка на нужную страницу, а во второй — текст, как она будет отображаться
`\href{https://ru.overleaf.com/}{удобный сайт}`

#### Ковычки
Из названия понятно что с привычными нам кавычками `"` в LaTeX возникают проблемы.

Основным вариантом решения данной проблемы является использование ` `` ` (совпадает с буквой "ё" на русской раскладке клавиатуры) в качестве открывающих кавычек и `''` (совпадает с буквой "э" на русской раскладке клавиатуры) в качестве закрывающих.

Чтобы использовать кавычки-ёлочки, необходимо ввести `<<` для открывающих кавычек и `>>` для закрывающих.

#### Структурирование

В классе документа `article`, в котором мы работаем, есть следующие команды для структурирования текста:
- `\section{название раздела}` — создание нового раздела;
- `\subsection{название подраздела}` — создание нового подраздела;
- `\subsubsection{название подподраздела}` — создание нового подподраздела.

Команда `\tableofcontents` **автоматически** создаёт **содержание** документа выбранном месте. Обращаю ваше внимание на то, что содержание является **кликабельным**, т.е. при нажатии на название раздела вы будете перемещены к месту в документе, где этот раздел находится.

Если вы по какой-то причине **не** хотите, чтобы раздел **был в содержании** документа, необходимо после команды \section, перед первой фигурной скобкой добавить символ `*`, т.е. `\section*{название раздела}`. Аналогично `\subsection*{название подраздела}` и `\subsubsection*{название подподраздела}` 

Если вы хотите добавить раздел, подраздел или подподраздел в содержание документа с ссылкой на местонахождение данной команды, то вы можете воспользоваться следующими командами:

- `\addcontentsline{toc}{section}{название добавляемого раздела}`;
- `\addcontentsline{toc}{subsection}{название добавляемого подраздела}`;
- `\addcontentsline{toc}{subsubsection}{название добавляемого подподраздела}`.

С помощью комбинации команд `\section*{название раздела}` и `\addcontentsline{toc}{section}{название добавляемого раздела}` вы можете добавить в содержание документа раздел, название которого будет отличаться от его названия в тексте.
