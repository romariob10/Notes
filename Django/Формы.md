***Форма*** - визуальный элемент, расположенный на web-странице и преднозначенный для ввода данных

#### Типы полей:
- checkbox - флажок
- file - поле для отправки файла
- text - ввод текста
- password - ввод пароля
- number - ввод чисел
- date - ввод даты
- email - ввод email-адреса
- hidden - невидимое поле

#### Виды форм:
- GET - запрос на получение данных с сервера
- POST - запрос на запись данных на сервер
