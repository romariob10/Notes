Во время создания проекта вы можете часто использовать повторяющиеся элементы HTML, и для уменьшения количество кода, который вам нужно писать, можно использовать шаблоны.


## 1. Создание шаблонов

Для удобства создайте файл шаблона, который будет содержать повторяющийся блок HTML-кода. Например, шаблон может содержать блок шапки сайта, блок меню, блок контента и блок подвала сайта.

Затем вы можете добавить изменяемые элементы в шаблон, например обозначение меню сайта, какой пункт открыт. Для этого в шаблоне напишите

``` HTML 
{% block name_of_block %}
{% endblock %}
```
##### ВАЖНО! Блоки не могут иметь повторяющиеся названия

## 2. Импорт шаблонов

В файле, в которой мы хотим импортировать блок пишим

``` HTML
{% extends 'nameofapplication/name_template.html' %}
```

Для изменения изменяемых элементов шаблона добвьте

``` HTML 
{% block name_of_block %}
<h1>Сюдо вписываем, что вам необходимо<\h1>
<h2>Можно и много кода<\h2>
{% endblock %}
```

## 3. Вставка другого кода

``` HTML
{% incude 'nameofapplication/name_template.html' %}
```

## 4. Загрузка статических файлов

``` HTML
{%.load static %}
```

Пример:
``` HTML
<link rel="stylesheet" href="{% static 'main/css/main.css' %}">
```

## 5. Ссылки на страницы
Вместо использование абсолютных ссылок принято использовать это:

``` Python
urlpatterns = [
	path('', views.index, name='mainpage'),
	path('about', views.about, name='aboutpage')
]
```

``` HTML
{% url 'mainpage' %}
```

## 6. Передача данных в HTML из render

На данный момент вывод HTML страниц выглядит так:
``` Python
def index(request):
	return render(request, 'main/index.html')
```

Но как передать из render данные в HTML?
Для этого необходимо, как следующий параметр предать словарь

Пример:
``` Python
def index(request):
	return render(request, 'main/index.html', {"title" : "Главная страница"})
```

Для получения данных мы пишем
``` HTML
{{ title }}
```

## 7.  Перебор значений

В случае если мы попробуем вывести массив, у нас выведится `['v1', 'v2', 'v3]`.  Для перебора значений используется синтаксис, схожий с Python:
``` HTML
{% for val in values %}
<li> {{val}} <\li>
{% endfor %}
```

## 8. Получение значения словаря по ключу

``` HTML
{{ dict.key }}
```

## 9.  Условия

В случае если мы попробуем вывести массив, у нас выведится `['v1', 'v2', 'v3]`.  Для перебора значений используется синтаксис, схожий с Python:
``` HTML
{% for val in values %}
{% if val > 15 %}
<li> {{val}} <\li>
{% endif %}
{% endfor %}
```
## 10.  Фильтры

Если а=при выводе необходимы фильтры, есть предустановленные в jinja (см. [[Jinja filters]])

Пример:
``` HTML
{% filter upper %}
{{ val }}
{% endfilter %}
```
