### ***ЗАДАЧА.*** У нас есть $N$ отдельных элементов. Нам необходимо отвечать на запросы вида:
- Создать новую группу из элемента $A_{i}$, где $i < N$.
- Определить в какой группе элемент $A_{i}$, где $i < N$.
- Объединить 2 группы.
---
### ***РЕАЛИЗАЦИЯ.*** Создадим дерево для группы, при объединение добавим дерево к другому дереву. 
- По умолчанию каждая вершина будет единственным элементом своего дерева.
- Создадим функцию `getp(int A)` для элемента $A_{i}$, где $i < N$, которая будет идти к корню и вернет номер группы. 
- Создадим функцию `union(int A, int B)`, которая проверит, что элементы в разных деревьях, если это так добавим корню 2го дерева корень 1го дерева, как родительский элемент.
---
### ***ДОПОЛНИТЕЛЬНО.*** 
- Храним для корня размер поддерева
