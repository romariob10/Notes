***Задача:*** посчитать кол-во чисел, чтобы соседние цифры не равны

``` Pseudo
n = input()
dp[n + 1][10]

dp[0].fill(1)
dp[0][0] = 0

for i = 1 … n - 1
	for j = 0 … 9
		for k = 0 … 9
			if j != k and abs(j - k) != 2
				dp[i][k] += dp[i - 1][j]
```
