#### Храним многоугольники как последовательность точек.
При задании многоугольника точки могут перечислятся, как по часовой стрелке, так и против.
[[Вектора]]
```cpp
//Описание класса вектор (см. ссылку выше)

//n - кол-во вершин
vector<Vector> polygon;

for (int i = 0; i < n; i++) {
	cin >> x >> y;
	polygon.emplace_back(x, y);
	//emplace_back потому что запускает конструктор
}
```

---
**Выпулый многоугольник** - многоугольник, все точки которого лежат по одну сторону от любой прямой, проходящей через две его соседние вершины
