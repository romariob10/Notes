### 1. Базовые определения и формулы

- Окружность: (x-x₀)² + (y-y₀)² = r²
- Центр: O(x₀, y₀)
- Радиус: r

### 2. Взаимное расположение окружностей

1. Не пересекаются: d > r₁ + r₂
2. Одна внутри другой: d < |r₁ - r₂|
3. Пересекаются: |r₁ - r₂| < d < r₁ + r₂
4. Касаются внутренне: d = |r₂ - r₁|
5. Касаются внешне: d = r₁ + r₂

где d - расстояние между центрами окружностей
