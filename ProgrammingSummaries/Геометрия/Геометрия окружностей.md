#### 1. Обозначения
- **Окружность**: $(x - x₀)² + (y - y₀)² = r²$
- **Центр**: $O(x₀, y₀)$
- **Радиус**: $r$
#### 2. Взаимное расположение окружностей 1. 
1. **Не пересекаются**: $d > r₁ + r₂$. 
2. **Одна внутри другой**: $d < |r₁ - r₂|$
3. **Пересекаются**: $|r₁ - r₂| < d < r₁ + r₂$
4. **Касаются внутренне**: $d = |r₂ - r₁|$
5. **Касаются внешне**: $d = r₁ + r₂$ где  $d$ — расстояние между центрами окружностей. 
#### 3. Касательные к окружности Для точки \(P(x_p, y_p)\) и окружности \(O(x₀, y₀)\): 


#### 4. Важные алгоритмы - **Проверка принадлежности точки окружности**: 

```cpp 
bool isOnCircle(Point p, Circle c) { 
double d = sqrt(pow(p.x - c.center.x, 2) + pow(p.y - c.center.y, 2)); 
return abs(d - c.r) < EPS; } 
```
