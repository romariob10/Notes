## Для начала прочти [[СНМ]]
---
### ***ЗАДАЧА.***  В графе нужно удалить ребра, чтобы сумма оставшихся весов была минимальна и можно было попасть из любой вершины в любую.

Очевидно, что ответ - дерево, потому что если есть цикл, можно удалить ребро, и получить лучший ответ.

Такая задача - задача на поиск ***минимальной остовы***.
***Остовы*** - Поддерево минимального веса во взвешенном графе.

---
### ***РЕШЕНИЕ.*** 

- #### Лемма о безопасном ребре
	Рассмотрим произвольный разрез (удалили некоторые рёбра так, что граф распался на две части) какого-то подграфа минимального остова. Тогда ребро минимального веса, пересекающее этот разрез является безопасным.
- #### Идея
	Получается, что мы можем действовать жадно — на каждом шаге добавлять ребро минимального веса, которое увеличивает наш остов.

### Алгоритм Прима
Один из подходов — строить минимальный остов постепенно, добавляя в него рёбра по одному.

- Изначально остов — одна произвольная вершина.
- Пока минимальный остов не найден, выбирается ребро минимального веса, исходящее из какой-нибудь вершины текущего остова в вершину, которую мы ещё не добавили. Добавляем это ребро в остов и начинаем заново, пока остов не будет найден.

Совсем наивная реализация за $O(N*M)$ — каждый раз перебираем все рёбра, но можно написать за $O(N^2)$

``` C++
#include <bits/stdc++.h>
using namespace std;

const int inf = 1e9;
vector<bool> used;
vector<vector<pair<int, int>>> g;
vector<int> min_edge, best_edge;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    
    used.resize(n, false);
    min_edge.resize(n, inf);
    best_edge.resize(n, -1);
    g.resize(n);
    
    for (int i = 0; i < n; i++) {
        int u, v, w;
        cin >> u >> v >> w;
        g[u].push_back({v, w});
        g[v].push_back({u, w});
    }

    min_edge[0] = 0;
    for (int i = 0; i < n; i++) {
        int v = -1;
        for (int u = 0; u < n; u++)
            if (!used[u] && (v == -1 || min_edge[u] < min_edge[v]))
                v = u;

        if (v == -1) break;
        
        used[v] = true;
        if (v != 0)
            cout << v << " " << best_edge[v] << endl;

        for (auto e : g[v]) {
            int u = e.first, w = e.second;
            if (!used[u] && w < min_edge[u]) {
                min_edge[u] = w;
                best_edge[u] = v;
            }
        }
    }

    return 0;
}
```
