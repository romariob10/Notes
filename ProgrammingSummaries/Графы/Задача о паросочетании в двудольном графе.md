**Двудольный граф** - граф, вершины которого можно разделить на 2 множества, и ребро соединяет вершины строго из разных множеств.  
Паросочетание - набор рёбер в двудольном графе, где каждой вершине инцидентно не более 1 ребро.  
Чередующая цепочка - путь в двудольном графе, для любых двух соседних рёбер которого верно, что одно из них принадлежит паросочетанию MM, а другое нет.

```C++
bool dfs(int v){  
used[v] = 1;  
  
for (auto u : graph[v]){  
if (to[u] == -1 || dfs(to[u]) && !used[v]){  
to[u] = v;  
return 1;  
}  
}  
  
return 0;  
}
```
