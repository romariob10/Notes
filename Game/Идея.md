Идеи игры состоит в прохождении многоуровнего подземелья с различными уровнями сложности.

Каждый уровень содержит различные задачи, которые нужно решить, чтобы пройти дальше.

Вначале игрок создает персонажа, выбирая расу, пол, имя и класс персонажа.

После создания персонажа игрок должен закупить ресурсы на стартовый капитал, но он может войти в долги.

В случае смерти на первых уровнях игрока воскрешают на он платит деньги, дальше - хуже, необходимо ждать, а также ты не можешь просто открыть магазин, придется заново спускаться (подем по заклинание => мгновенен).

При прохождении из КАЖДОГО монстра выпадают ВСЕ ресурсы, чтобы обойтись без магазина.
