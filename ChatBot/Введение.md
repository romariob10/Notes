##### *API* (Application Programming Interface — программный интерфейс приложения) — это набор способов и правил, по которым различные программы общаются между собой и обмениваются данными. 

*Используется разработчиками при написании
*Типизация*
Механизм языка определяющий тип переменных

Типы данных

| *Динамическая*                                                                             | *Статическая*                                                                                                               |
| ------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- |
| определяются во время выполнения кода, при создании переменных указывать его не надо | Типы данных определяются на этапе компиляции, тип обязательно указывается при объявлении переменной и больше не изменяется. |
