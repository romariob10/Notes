### _Проблема_

Существующие AI-дизайнеры (v0, Bolt, Lovable, Claude Design) хорошо выглядят на лендингах с воздухом, но «глаза плавятся» в data-dense приложениях (dashboard, CRM, admin, BI). Они генерируют токены на лету, не зная про row-height, density-modes, state-цвета, focus-ring. Публичные дизайн-системы (Linear, Stripe, Vercel) уже есть, но разбросаны, в разных форматах, и ни один AI-агент не умеет их подбирать под контекст задачи. Рынок design tokens — $X млрд, рынок AI-builder'ов — миллионы юзеров, но прослойки «agent + curated DS + lint» не существует.

---
### ***Решение***

**«Design System Marketplace + skill для AI-агентов».**
- Бэкенд = каталог курированных мини-DS (палитра, типографика, радиусы, тени, spacing, density, state-цвета), категоризированных не по бренду, а по use-case: SaaS-dense, CRM power-user, internal tool, dev-tool IDE-like, marketplace, mobile-first.
- Скилл для Codex/Antigravity/Hermes/OpenClaw — устанавливается одной командой. Когда юзерский агент генерит UI, скилл подбирает подходящую DS, встраивает токены, прогоняет линтер (контраст, density, state-coverage, focus-ring), отдаёт ссылки на твой каталог.
- Free tier: 3-5 открытых DS (Vercel/Geist, Salesforce/Lightning, IBM/Carbon под MIT/Apache), без линтера.
- Pro tier $19/мес: 20+ курированных DS, design-lint, экспорт React/Vue/HTML, импорт токенов из JSON/Style Dictionary/Figma Variables.
- Ключевое отличие: дизайн не выдумывается нейросетью, а собирается из проверенных use-case-паттернов → одинаково хорошо и на лендинге, и в admin-панели с 30 колонками.

---
### ***Заработок***

- **Free + Pro подписка $19/мес** — основной recurring. Конверсия 3-5% с free.
- **Per-call API $0.05-0.20/lookup** — для встраивания в чужие агенты через маркетплейсы (Codex, Antigravity, Open Design).
- **Разовая продажа DS-пакетов $499-2999** — по модели Untitled UI (Studio/Business).
- **Enterprise $5K-30K/год** — кастомная DS-категория под бренд заказчика, on-premise.
- **Dual-license:** MIT на free, проприетарная на Pro. Код, сгенерированный с твоей DS, принадлежит юзеру.
