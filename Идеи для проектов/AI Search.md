**Суть:** онлайн копия Obsidian, но с удобной возможностью структурировать файлы через ИИ
**MVP:** ИИ обрабатывает заметки и создает облако тэгов, по облакам тегов идет кластеризация. Выход - граф мыслей. Дополнительно - поиск, который по тексты идет от большего кластера вглубь, понимаю идею, а не просто ищя слова
**Киллер фичи:** Работа нескольких людей одновременно. Дополнение в браузер для синхранизации контента и мыслей
**Сложный доп:** Совместимый API с Obsidian для использования расширений без переписывания
